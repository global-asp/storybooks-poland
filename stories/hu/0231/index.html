Száraz az idő.

Jest sucho.

It is dry.

Forrón süt a nap.

Słońce jest gorące.

The sun is hot.

Szeles az idő.

Jest wietrznie.

It is windy.

Felhős az ég.

Jest pochmurno.

It is cloudy.

Hideg van.

Jest zimno.

It is cold.

Esik az eső.

Pada deszcz.

It is raining.

Vihar van.

Grzmi.

There is thunder.

Látok egy szivárványt.

Widzę tęczę.

I see a rainbow.

Written by: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
