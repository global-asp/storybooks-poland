Aš galiu bėgioti.

Dzięki niemu mogę biegać.

I can run with it.

Aš galiu šokinėti.

Dzięki niemu mogę skakać.

I can jump with it.

Aš galiu šokti.

Dzięki niemu mogę tańczyć.

I can dance with it.

Aš galiu plaukti.

Dzięki niemu mogę pływać.

I can swim with it.

Aš galiu šokčioti.

Dzięki niemu mogę przeskakiwać.

I can skip with it.

Aš galiu spirti.

Dzięki niemu mogę kopać.

I can kick with it.

Aš galiu pabėgti.

Dzięki niemu mogę uciekać.

I can escape with it.

Bet aš niekada negalėsiu skristi.

Ale nigdy nie będę mógł latać.

But I can never fly with it.

Written by: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
Illustrated by: Mlungisi Dlamini, Ingrid Schechter
Translated by: Lina Baltrukoniene
Language: Lithuanian
